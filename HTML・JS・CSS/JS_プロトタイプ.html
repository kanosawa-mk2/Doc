プロトタイプ

Javascriptにはクラスがなくプロトタイプ「ひな形」が存在する。

Javascriptのクラス定義
var Member = function(){};

インスタンス化
var mem = new Member();

Javascriptでは関数(Functionオブジェクト)にクラスとしての役割を与えている

コンストラクタ
オブジェクトを生成することを想定した関数オブジェクト

    var Member = function (firstName, lastName) {
        this.firstName = firstName;
        this.lastName = lastName;
        this.getName = function () {
            return this.lastName + ' ' + this.firstName;
        }
    };
    var mem = new Member('四方田', '春海');
    document.writeln(mem.getName());
↓実行結果↓

クラスのプロパティとメソッド

  • プロパティ　→　値がオブジェクト
  • メソッド　　→　値が関数オブジェクト

上記の例では、firstNameがプロパティ、getNameがメソッド
